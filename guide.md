# Guide til webutvikling

## Hvem, hva og hvorfor?

## header-elementet

- Logo
- Søkemotor
- Logg inn / Register
- Navigasjon

## main-elementet:

- Hero med viktigest info
  - Info om hva siden er
  - Spesielle tilbud
  - Nytt som skjer
- Tilegg seksjoner med mer info
  - Kort med informasjon
  - Dypere navigasjon
  - Reklame for siden/produkt

## footer-elementet:

- Logo
- Navigasjon
- Personvernserklæring / TOS
- SoMe
- Kreditasjon
